🚙 Planck Racer Help

✨ Features

  • Planck.js vehicle with springy wheel joints, motor/brake torque, and in‑air pitch control.
  • Procedural terrain (sinusoidal blend) with slope limiting, culling & on‑the‑fly generation.
  • HUD metrics: Speed (km/h), RPM, Fuel %, Distance (m), Slope %, Angle (°) & Applied Torque.
  • Checkpoints & fuel cans automatically placed on peaks.
  • Game states: Pause/Resume, Debug overlay, Game Over on fuel depletion.
  • Full input support: Keyboard, Gamepad, and Mobile touch controls.
  • Stable physics: 120 Hz fixed-step simulation with an accumulator.

🧪 Dev/Debug Shortcuts

  • H — Toggle this Help panel
  • D — Toggle Debug visuals (joint anchors, different HUD color)
  • Space — Pause/resume the simulation
  • R — Reset to the last checkpoint (spawns slightly above ground)
  • Out of bounds (falling too far down) auto-resets the car.
